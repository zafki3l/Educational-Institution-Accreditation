# Educational-Institution-Accreditation

# Project Setup

## 1. Requirements
- PHP 8.2+
- Composer

## 2. Install
Clone project:
```bash
git clone https://github.com/zafki3l/Educational-Institution-Accreditation.git
cd Educational-Institution-Accreditation
```

Setup composer:
```bash
composer install
```

Create .env file:

```bash
cp .env.example .env
```
